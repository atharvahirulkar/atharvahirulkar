## 🎯 Featured Projects

### [MarketPulse](https://github.com/atharvahirulkar/marketpulse) - Real-Time Market Data Pipeline
Multi-source streaming architecture with sub-100ms end-to-end latency  
`Polygon.io WebSocket` → `Kafka` → `PySpark Streaming` → `TimescaleDB` → `Grafana`  
⚡ **>1000 ticks/sec** · 📊 Full local Docker stack · Production-ready monitoring

### [FraudAPI](https://github.com/atharvahirulkar/fraud-detection) - ML Fraud Detection API  
Ensemble gradient boosting with comprehensive MLflow experimentation  
`IEEE-CIS dataset` → `XGBoost + LightGBM` (20+ experiment runs) → `FastAPI` → `GCP Cloud Run`  
🎯 **AUC-PR >0.88** · ⚡ **p99 latency <50ms** · Production ML registry

### [CosmeTik](https://github.com/atharvahirulkar/cosmetik) - Hybrid Vector + Graph ML Recommender
Multi-database intelligence: relational + graph + vector embeddings  
`PostgreSQL (3NF)` + `Neo4j (PageRank)` + `Qdrant (384-dim embeddings)` → `Streamlit UI`  
📦 **8.5K products** · ⭐ **1M+ reviews** · 🔗 **30 ingredient communities detected** · ML-powered discovery

### [PulseML](https://github.com/atharvahirulkar/pulseml) - IoT Wearable Biosensor System
Full-stack IoT → signal processing → anomaly detection  
`Custom ECG/PPG wristband (MAX86150 + Arduino Bluno BLE)` → `HeartPy signal processing` → `LSTM classifier`  
🏆 **Copyright Reg. No. L-114951/2022** (Govt. of India) · Arrhythmia detection · HRV analytics · BPM forecasting

### [Multilingual Speech Transcription](https://github.com/atharvahirulkar/multilingual-speech-transcription) - Video-to-Text Pipeline
End-to-end video processing with multi-stage NLP  
`Google Speech Recognition` + `BART-Large summarization` + `Google Translate` → `Flask web app`  
🌍 **100+ languages** · 📝 **~95% WER (English)** · Accessible transcription at scale
